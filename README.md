# WMS Product Management System

상품 조회 및 반입 및 반출 관리 기능을 제공하는 간단한 WMS(Web-based Warehouse Management System) 프로젝트입니다.  
---

## 1. 주요 기능



### 상품 관리
- 전체 상품 조회
- 키워드 검색, 컬럼 기준 정렬 (ASC / DESC), 페이징 처리
- 상품 등록
- 상품 수정
- 상품 삭제
- 반입
- 반출

---

## 2. 기술 스택

### Backend
- Java 17
- Spring Boot
- MyBatis
- PostgreSQL
- Gradle 

### Frontend
- React
- TypeScript
- Tailwindcss
- Vite
- Axios
